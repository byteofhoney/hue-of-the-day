## 🤝 Contributing

This project is open for contributions! Due to limited time on my end, I would like your help growing it further.

**The idea:** think of it as a *Merriam-Webster, but for colors* every day, the app serves up a "Color of the Day" with its hex code, along with a matching flower recommendation that shares a similar hue.

Feel free to fork, open issues or submit PRs whether it's new features, bug fixes, UI improvements or better color/flower matching logic.

### Ways you can help
- 🎨 Improve color matching accuracy
- 🌸 Expand the flower database
- 🐛 Fix bugs or improve error handling
- 💅 Polish the UI/UX
- 📝 Improve documentation

To contribute:
1. Fork the repo
2. Create a new branch (`git checkout -b feature/your-feature`)
3. Commit your changes
4. Open a pull request
